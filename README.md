# MongoDB-in-Python

The purpose of this project is to provide strategies for a food app to increase numbers of users. Using the public MongoDB NoSQL database, key recommendations are:

- In the front page, inform consumers the Top 5 most popular cuisine types in New York.
- Put the information of the Top 10 best-scored American and Chinese restaurants and ensure consumers can see it when they open the app.
- Also list the Top 10 best restuarants in Manhattan and Brooklyn for customers.

[Please click this link for the report and scripts](https://github.com/Mengrou-Wang/MongoDB-in-Python/blob/main/Mongo%20DB%20in%20Python.ipynb)
